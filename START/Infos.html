About

This website is created by the researchers from Rizal High School Arts and Design strand. This purpose is to teach Arts and Design students color theory in an elaborate way while being able to apply them in real-time. This website consists of information related to the color theory that artists apply in their artworks.

Main sections of this website are dictionary, introduction, color combinations and application. The interactive elements of this websites are a match the color, color wheel, color mixing, search bar, buttons, links, pictures and an avatar (Hue).

Interactive Elements

The interactive elements can be found throughout the website. Common elements include images, buttons, navigation bar and links. Certain elements can only be found inside the lessons, application, references and dictionary.
